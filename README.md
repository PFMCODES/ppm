## How to install
### npm:
<pre>

</pre>

### yarn:
<pre>

</pre>

### pnpm:
<pre>

</pre>

<div id="init">

### init
<pre>
<span style="color: #06B8B8">ppm</span> init</pre>

> initializes a new package

<pre>
<span style="color: #06B8B8">ppm</span> init -y</pre>

> initalizes a new package with default settings

<div>
<div id="install">

### install
<pre>
<span style="color: #06B8B8">ppm</span> install [package name]</pre>

<pre>
<span style="color: #06B8B8">ppm</span> install [package name] -g</pre>
<div>

> installs a package globally or loacally
